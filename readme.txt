== Description ==

A widget showing your latest video status uploaded to your http://Robo.to wall. Simple!

All you need is to find your UUID code from your Robo.to account, paste it in within wp-admin widget settings, and you're sorted!

For a demo, see this site: http://blog.iandundas.co.uk/2009/08/robo-to-wordpress-widget/

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `roboto.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Done!
